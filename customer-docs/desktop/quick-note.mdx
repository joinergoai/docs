Use this page when you want a desktop recording linked to a note instead of relying only on calendar detection.

## Who can use this

- Users capturing an ad-hoc meeting or conversation.
- Operators troubleshooting recordings that should link back to a note.

## Before you start

- Sign in to Ergo Desktop.
- Grant macOS capture permissions.
- Create or open the note you want the recording to relate to, if your workflow starts from a note.

## Steps

- Start the Quick Note or manual-note recording flow.
- Confirm recording is active.
- Add or edit the note title while the recording is in progress if needed.
- Stop recording when the conversation ends.
- Wait for upload and processing.
- Confirm the processed meeting or recording is linked to the note.

## What to expect

- Desktop recordings can carry a linked note ID when the recording starts from a note.
- If the note title is more specific than the recording-time title, Ergo can use the linked note title.
- Untitled notes may be auto-titled after processing.

## Common issues

- The recording was started outside the note flow, so it is not linked to the note.
- The note title was still generic when processing completed.
- Upload or processing failed before the note link could be updated.
