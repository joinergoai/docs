Use this section when you are recording through the Ergo Desktop app instead of relying only on the cloud notetaker bot.

## Who this section is for

- Users recording meetings locally from their desktop.
- Admins and operators helping users verify install, sign-in, permissions, detection, recording, upload, and processing.

## Before you start

- Confirm your workspace has access to Ergo Desktop.
- Install the app and sign in with the same Ergo account you use in the dashboard.
- Grant macOS permissions before relying on local capture.
- Run a short test capture before important calls.

## Common issue path

- If setup fails, start with sign-in and macOS permissions.
- If capture starts but the meeting is missing, check recording status, upload state, and processing state.
- If detection is unreliable, use a manual recording or Quick Note workflow as a fallback.
