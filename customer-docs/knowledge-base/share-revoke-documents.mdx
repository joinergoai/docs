Use document sharing when a generated document should be accessible through a share link.

## Share a document

- Open the generated document.
- Use the share action to create a share token.
- Copy the shared URL and send it only to intended recipients.

## Revoke a document link

- Use the revoke share action on the document.
- Revoking removes the share token for that document.
- Revoking a document link does not delete the document.

## Access notes

- Shared document viewing is separate from internal document visibility.
- Download access follows the shared or authenticated document route being used.
- If a document is linked to deals, review linked deals before changing visibility or deleting it.
