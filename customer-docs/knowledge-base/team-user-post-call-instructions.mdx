Team and personal instructions are layered so shared guidance can coexist with each user's writing preferences.

## Team instructions

- Admins can select a team and save custom instructions for that team.
- Team instructions are shown read-only to non-admin users.
- Team instructions are added only to post-call drafts for members of that team.

## User instructions

- Each user can save personal instructions.
- Personal instructions are autosaved after editing and also saved on blur.
- Personal instructions apply only to that user's post-call drafts.

## Best practices

- Put company-wide rules in org instructions.
- Put role or segment guidance in team instructions.
- Put tone, sign-off, and personal follow-up preferences in user instructions.
- Keep instructions current when templates or sales motions change.
