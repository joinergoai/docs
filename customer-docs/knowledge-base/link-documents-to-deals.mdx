Link documents to deals when a generated document should appear with the relevant deal context.

## Link a document

- Open the generated document or document list.
- Choose the deal-linking action.
- Search for the deal.
- Select one or more deals and save.

## Manage linked deals

- Review the linked deal count before changing visibility.
- Remove all deal links when the document should no longer appear in deal context.
- Use the deal right rail to review documents already linked to a deal.

## Caveats

- The checked deal-linking API applies to generated documents.
- Uploaded knowledge base files are reference context and should not be described as structured deal attachments unless the current UI explicitly supports that workflow.
