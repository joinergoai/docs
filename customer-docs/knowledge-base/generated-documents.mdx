Generated documents are output files created by Ergo workflows, separate from uploaded knowledge base files.

## What you can manage

- Search and filter generated documents.
- Open or preview a generated document.
- Download the generated file.
- Update title, tags, or visibility where the UI exposes those controls.
- Delete generated documents you no longer need.

## Visibility states

- **Private**: visible to the creator or authorized users.
- **Org**: visible more broadly inside the org where supported.
- **Shared**: available through a share token.

## Important distinction

Uploading a knowledge base file gives Ergo reference context. A generated document is an output created by Ergo. Manage their upload, indexing, sharing, and deletion states separately.
