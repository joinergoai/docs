Post-call custom instructions tell Ergo how to write follow-up drafts after meetings.

## Instruction layers

- **Org instructions** apply to post-call drafts across the org.
- **Team instructions** apply to drafts for members of the selected team.
- **My instructions** apply only to your own post-call drafts.

## How admins use it

- Open post-call configuration.
- Choose **Entire org** or a specific team.
- Edit custom instructions and save.
- Add reference documents when the draft should use a playbook or other reusable guidance.

## How users use it

- Review read-only team guidance where available.
- Add personal preferences in **My Instructions**.
- Keep personal instructions specific, such as tone, formatting, links, or follow-up style.
