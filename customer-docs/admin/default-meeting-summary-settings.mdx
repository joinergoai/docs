Use this page to set the default post-call summary email behavior for users in the organization. This controls whether summary emails are enabled by default and whether those emails include a recording and transcript link.

## Who can use this

- Super admins can set the organization-wide default from the calendar settings menu.
- Individual users can still change their own post-call summary email setting after the default is applied.

## Before you start

- Decide whether post-call summary emails should be on by default for the organization.
- Decide whether summary emails should include a **View Recording & Transcript** link by default.
- Tell users if you are changing the default before a rollout.

## Steps

1. Open the calendar settings menu.
2. Find **Send Summary Email**.
3. Turn the switch on or off.
4. If summary emails are on, choose whether **Send Recording & Transcript** should also be on.
5. Save and test with one eligible meeting.

## What to expect

- **Send Summary Email** sets the default for post-call summary emails.
- **Send Recording & Transcript** controls whether the summary email includes the recording/transcript button by default.
- The setting updates the user's notetaker configuration. It does not change meeting visibility, reporting access, or recording-disclaimer behavior.
- A summary email is only sent when Ergo has the required meeting data and generated summary. Some meetings may not produce or send a summary email even when the default is enabled.
- Private meetings do not include the recording link in the summary email.

## Common issues

- **The setting is missing.** Confirm the acting user is a super admin.
- **A user changed it back.** Users can change their own post-call summary email setting after the default is applied.
- **No email was sent.** Confirm the meeting was processed, had a generated summary, had valid attendees, and met the delivery requirements.
- **The recording link is missing.** Confirm **Send Recording & Transcript** is enabled and the meeting is not private.
