Use this section when you know the person's job but do not yet know which product area or setup page they need. Role playbooks are curated paths through the docs; they show what each person can get out of Ergo, then point back to the setup, workflow, troubleshooting, and reference articles.

## Choose a role

- [RevOps playbook](/customer-docs/roles/revops): Keep pipeline data trustworthy, configure mapped CRM updates, and reduce audit work.
- [Sales manager playbook](/customer-docs/roles/sales-manager): See which deals, reps, meetings, and follow-ups need attention.
- [AE playbook](/customer-docs/roles/ae): Know what to work on, ask deal questions, generate reviewable follow-up drafts, and keep mapped CRM fields updated when configured.
- [CSM playbook](/customer-docs/roles/csm): See full account history, spot renewal or expansion signals, and draft account outreach.
- [SDR playbook](/customer-docs/roles/sdr): Prioritize outreach, prepare handoffs, draft follow-up, and use context from meetings, email, and CRM.

## If you are setting up a workspace

- Start with [First-time setup checklist](/customer-docs/setup/first-time-setup-checklist).
- Use [Setup by role](/customer-docs/setup/setup-by-role) to decide who owns shared setup versus personal setup.
- Use [RevOps setup for Ergo rollout](/customer-docs/setup/revops-setup-for-ergo-rollout) when one owner is coordinating the launch.

## If something is broken

- Start with [Troubleshooting](/customer-docs/troubleshooting/index).
- Use [CRM sync issues](/customer-docs/troubleshooting/crm-sync-issues) for missing fields, stages, records, or CRM updates.
- Use [Notetaker did not join](/customer-docs/troubleshooting/notetaker-did-not-join) or [Transcript or recording missing](/customer-docs/troubleshooting/transcript-or-recording-missing) for capture issues.
- Use [Draft send failures](/customer-docs/troubleshooting/draft-send-failures) or [Duplicate drafts from multiple notetakers](/customer-docs/troubleshooting/duplicate-drafts-from-multiple-notetakers) for draft issues.
