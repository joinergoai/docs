Use this page when RevOps, sales operations, GTM operations, or an admin owner is preparing Ergo for a team. The goal is to make the shared setup reliable before users judge meetings, drafts, CRM updates, reporting, or AI output.

## What RevOps usually owns

- CRM connection, source-of-truth decisions, field mapping, pipeline stages, and stage controls.
- Shared setup for users, teams, templates, notetaker defaults, reporting defaults, and access.
- Rollout sequencing, pilot verification, first-week issue routing, and support escalation context.
