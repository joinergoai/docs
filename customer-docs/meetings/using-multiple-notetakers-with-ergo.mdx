Using more than one notetaker can be useful, but it can also create duplicate meeting records and duplicate downstream work.

## What can duplicate

- Meeting history records.
- Transcripts or summaries.
- Post-call drafts.
- CRM updates.
- Reporting and automation outputs.

## How to reduce duplication

- Pick one primary capture source for each recurring workflow.
- Turn off the Ergo bot for meetings where another notetaker should be the source of truth.
- Check the meeting source before pushing automations.
- Review drafts before sending when more than one recorder joined the same meeting.

## When duplication is expected

- You use both the Ergo bot and a third-party notetaker on the same call.
- You upload an external recording of a meeting already captured by the bot.
- You create manual notes and also process a transcript for the same event.
