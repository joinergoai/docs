Recurring meetings need an extra decision: whether you are changing one occurrence or the future series.

## Schedule recurring meetings

- Turn on the notetaker for the occurrence you want to capture.
- If Ergo prompts for a recurring action, choose the option that matches the intended scope.
- Recheck the next occurrence if the series has exceptions or changed links.

## Cancel recurring meetings

- Choose **this only** when a single occurrence should skip the notetaker.
- Choose **all future** when future occurrences should not be scheduled.
- Review already scheduled future events, because existing notetaker records may need to be canceled as part of the series change.

## Caveats

- Calendar providers represent recurring series differently, so a single changed occurrence can have a different event ID from the series.
- If the meeting URL is reused across a recurring series, verify the specific occurrence before assuming the whole series is configured.
- Calendar sync, provider scheduling, and dashboard display can update at different times.
