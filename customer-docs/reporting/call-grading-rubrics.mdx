Call grading rubrics define scored criteria that Ergo can use in Reporting. Use them when managers need repeatable coaching views rather than ad hoc meeting notes.

## Who can use this

- Users with Reporting create access can manage rubrics.
- Rubrics can be organization-level or scoped to a team.

## Build a rubric

1. Open **Reporting**.
2. Select **Call Grading**.
3. Add a rubric at the organization level or inside a team section.
4. Enter a rubric name and description.
5. Add meeting title phrases if the rubric should target only certain meeting types.
6. Add rubric fields.
7. For each field, define grading options with scores and labels.
8. Save or let autosave complete before using the rubric in charts.

Rubrics require at least one rubric field before they can be saved.

## AI and default rubric options

- The AI rubric builder can draft a rubric from a prompt.
- AI can also help draft an individual rubric field.
- A default rubric can be added when available.
- Existing rubrics can be copied, including to team scope when supported.

Review generated fields and scoring labels before using them for coaching.

## Use rubrics in Reporting

Rubrics appear as reporting categories. Rubric charts can show rep-level grading patterns and can be filtered by supported fields. They differ from ordinary sentence or picklist fields, so some breakdowns and filters may be unavailable.

## Keep grading fair

- Keep scoring options explicit.
- Use conditional scoring only when managers understand when it applies.
- Pair rubric charts with source evidence before coaching.
- Update meeting title phrases if the rubric is catching the wrong meetings.
