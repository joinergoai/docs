Use shared Reporting links when someone needs to review a chart or dashboard outside the normal authenticated Reporting page.

## Chart link or dashboard link

- **Chart links** share one widget.
- **Dashboard links** share the dashboard view.
- Each link can be password protected.
- Each link can be revoked by unsharing it.

Shared links are best for a specific review moment. For ongoing internal access, use Reporting access grants instead.

## Share a chart

1. Open the dashboard that contains the chart.
2. Open the chart share action.
3. Copy the generated link.
4. Add a password if the audience needs one.
5. Send the link only to intended viewers.

## Share a dashboard

1. Open the dashboard.
2. Select **Share**.
3. Copy the generated dashboard link.
4. Add a password if needed.
5. Revoke the link when access should end.

## Password rules

- Passwords must be at least 6 characters.
- Passwords cannot exceed 128 characters.
- Updating the password changes the password required for that shared link.
- Removing the password leaves the shared link active unless you unshare it.

## Shared view limits

Shared views can be narrower than authenticated Reporting. Some sentence detail and revenue deal detail views are available only from authenticated charts. If a reviewer needs full drilldown access, grant them Reporting access instead of relying on a public link.
